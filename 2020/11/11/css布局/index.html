2.Flex布局

(1)盒模型

box占用的空间=content+padding+border+margin

(2)这里还要介绍两个属性，display 和 poistion

display：

  • block（元素表现为块级元素，有固定宽高，独占一行）
  • inline（元素表现为行内元素，不能设置宽高。高度就是内容文字或者图片的宽度，不可以改变。多个相邻的行内元素排在同一行里，知道页面一行排列不下，才会换新的一行。）
  • inline-block （对外表现为行内元素，对内表现为块级元素。使其既有block的宽度高度特性，又有inline的同行特性）

position：

  • static（默认情况，存在文档流当中）
  • relative（根据元素本身原来所应该处的位置偏移，不会改变布局的计算）
  • absolute（绝对定位，脱离文档流，不会对别的元素造成影响，相对的是父级最近的 relative 或者 absolute 定位元素）
  • fixed（绝对定位，脱离文档流，相对于的是屏幕，就是那些浮动的广告那样，怎么拉都固定在同一个位置，而 absolute 元素离开屏幕就看不见了）

(3) flexbox 布局（弹性盒子布局）

特点是盒子为并列的，只需要指定宽度。

E.g. 三栏布局（宽度均分）

三栏布局（两边宽度固定，中间自适应）

3.float 布局

(1)两栏布局

代码中最重要的是 margin-left: 100px;，这一句将浮动元素的位置空了出来，所以右边栏里面的元素不会影响到浮动的 div。这就是浮动布局的基本思想。

(2)三栏布局

注意：中间的元素要写在最后，因为中间元素假设有块级元素的话，会影响右边浮动元素的定位。

4.响应式布局

响应式布局指的是同一页面在不同屏幕尺寸下有不同的布局。

Bootstrap为经典的响应式布局框架

(1)使用媒体查询@media【IE6~8不支持】

@media 媒体类型and （媒体特性）{你的样式}

E.g.

  • 最大宽度max-width

“max-width”是媒体特性中最常用的一个特性，其意思是指媒体类型小于或等于指定的宽度时，样式生效。

  • 最小宽度min-width

“min-width”与“max-width”相反，指的是媒体类型大于或等于指定宽度时，样式生效。

  • 多个媒体特性使用
  • not关键词

使用关键词“not”是用来排除某种制定的媒体类型，也就是用来排除符合表达式的设备。换句话说，not关键词表示对后面的表达式执行取反操作。

(2)百分比布局

缺点：计算比较复杂。

(3)视口单位

vw表示相对于视图窗口的宽度，vh表示相对于视图窗口高度。1vw 等于视口宽度的1%，1vh 等于视口高度的1%。

(4)rem布局

rem单位都是相对于根元素html的font-size来决定大小的,根元素的font-size相当于提供了一个基准，当页面的size发生变化时，只需要改变font-size的值，那么以rem为固定单位的元素的大小也会发生响应的变化。 因此，如果通过rem来实现响应式的布局，只需要根据视图容器的大小，动态的改变font-size即可（而em是相对于父元素的）。

5.图片响应式

(1)使用max-width（图片自适应）

图片自适应意思就是图片能随着容器的大小进行缩放。

(2)使用srcset

如果屏幕的dpi = 1的话则加载1倍图，而dpi = 2则加载2倍图，手机和mac基本上dpi都达到了2以上，这样子对于普通屏幕来说不会浪费流量，而对于视网膜屏来说又有高清的体验。

如果浏览器不支持srcset，则默认加载src里面的图片。

(3)使用background-image

(4)使用picture标签（IE不支持）
